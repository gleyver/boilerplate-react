# Estrutura de Domínios

Cada domínio representa um contexto de negócio independente.

Exemplo:
- user (usuário)
- product (produto)
- order (pedido)

Cada domínio pode conter:
- components/
- pages/
- services/
- hooks/
- stores/
- types/
- tests/

Exemplo de estrutura:

src/domains/user/
  components/
  pages/
  services/
  hooks/
  stores/
  types/
  tests/
